# Navy PFA - Armada Ecuador

Aplicación móvil Flutter offline-first para autoevaluación, entrenamiento, consulta de reglamentos y asistencia IA.

## Módulos
1. Simulador (Calculadora de Baremos)
2. Centro de Entrenamiento (Videos)
3. Mi Rendimiento (Historial)
4. Salud y Nutrición
5. Chatbot Asistente (IA)

## Tecnologías
- Flutter
- sqflite
- chewie
- fl_chart
- Gemini/OpenAI API

## Estructura
- lib/modules/
- lib/models/
- lib/services/
- assets/videos/
- assets/images/
- assets/baremos.json

## Instalación
1. Instala Flutter SDK.
2. Corre `flutter pub get`.
3. Reemplaza los assets de ejemplo por los oficiales.

## Uso
- App offline, solo el chatbot requiere internet.
